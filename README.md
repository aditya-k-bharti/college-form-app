# College Form Application

A full-stack web app to collect and store college admission form data. Built using:

- Node.js
- Express
- HTML/CSS/JS
- MSSQL

## Features
✅ Dynamic form input  
✅ Server-side validation  
✅ MSSQL integration  
✅ Neat frontend with responsive design

## How to Run
```bash
npm install
node Server.js
